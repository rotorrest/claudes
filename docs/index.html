SYNOPSIS

claudios [-w sec]
snapshot, or watch mode refreshing every few seconds
claudios focus <id>
jump to that session's tab ‚ÄĒ Terminal/iTerm2 by tty, Cursor/VS Code/Windsurf by project window, across Spaces
claudios --json
machine-readable state for statuslines and scripts
claudios update
self-update from the latest release, every file SHA256-verified

DESCRIPTION

  • Sorted by attention. Blocked sessions first, then stopped ones ‚ÄĒ with Claude's last message so you know where it left off ‚ÄĒ then everything still working.
  • Background-agent aware. A session whose subagents are still writing shows as working, not idle. No more "wait, it wasn't done?"
  • Click-to-focus notifications. claude-notify turns Claude Code hooks into native macOS notifications; click one, land in that tab.
  • System headroom. CPU, RAM pressure, swap, per-container Docker, battery temperature, thermal throttling ‚ÄĒ can this Mac take one more session?
  • Token usage & burn rate. The last 5 hours of tokens and API-equivalent cost, read from your local JSONLs ‚ÄĒ press u for the per-session breakdown.

SECURITY

Everything is read locally. No telemetry, no accounts, no dependencies to audit ‚ÄĒ CI enforces stdlib-only imports and every release passes bandit + CodeQL before publishing. The one network call is a daily version check (CLAUDIOS_NO_UPDATE_CHECK=1 turns it off).

FAQ

Does it send my code or transcripts anywhere?
No. Everything is read locally from ~/.claude/. The one network call is a daily version check (CLAUDIOS_NO_UPDATE_CHECK=1 turns it off).
How is it different from ccusage or Claude-Code-Usage-Monitor?
Those track usage and cost. claude-monitor answers which session needs me right now and jumps you there ‚ÄĒ and since v0.3 it also shows your 5-hour token usage, burn rate and API-equivalent cost.
Does it work with Cursor or VS Code terminals?
Yes ‚ÄĒ it walks the process tree to find the host app: exact tab in Terminal/iTerm2 by tty, project window in Cursor/VS Code/Windsurf, across Spaces.
Why "Claudios"?
Because nobody runs just one Claude. Claudios is the plural you end up with.
